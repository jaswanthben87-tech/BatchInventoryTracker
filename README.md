# Batch Inventory Tracker

A web-based inventory management system for homemade food businesses.

## Features
- Batch-wise inventory tracking
- Manufacturing and expiry date management
- Product quantity monitoring
- Packaging count tracking
- Low stock alerts
- Inventory dashboard
- Search and filter batches
- Stock depletion monitoring

## Purpose
To help small food manufacturers efficiently manage production batches and inventory while reducing manual tracking through spreadsheets, calls, and messaging applications.
